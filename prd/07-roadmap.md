# 07. ロードマップ

> **状態: スケルトン**。本文は執筆予定。

## フェーズ（予定）

### MVP
- 認証（better-auth + Google OAuth）、private データ。
- 投入: ファイル/貼り付けインポート（自前 LLM）＋ スクショ証跡保存（各画面1枚）。
- レビュー&整合チェック → 確定。
- run 一覧/詳細、最小カタログ管理、記述的分析（スコア推移・頻度・タイミング分布）。
- ストレージ: ローカル `BlobStore`。self-host。

### Phase2
- MCP（`submit_run` / `get_extraction_guide` / `list_catalog`）＋ API（トークン）。
- 相関分析。カタログのリッチ属性（カテゴリ・色）。
- 必要ならクラウド移行（CF/Vercel、R2/S3 アダプタ）。

### Phase3
- サーバ側 LLM 全自動分析（worker、画像→構造化）。
- 課金（有料: 全自動）。
- 横断統計 / ランキング（オプトイン、自己申告スコアの信頼性対策）。
